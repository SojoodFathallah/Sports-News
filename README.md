# Champion News 

Champion news is a platform didecated to bringing up to date news about sports. It's main purpose is to keep you updated on anything sports related at all times.
Currently.

it contains two pages:

-Home Page
-Article Details Page

## Installation steps
npm create vite@latest sports-news

cd sports-news
npm install

## Libraries used
npm install axios
npm install @fontsource/roboto-condensed
npm install react-icons

## API used
news API.
